# 导入导出功能改进

## 修改内容

### 1. 新增组件 - ImportExportModal
- 文件：`src/components/ImportExportModal.tsx`
- 功能：显示导入模式选择弹窗
- 提供两个选项：
  - **覆盖导入**：删除现有任务，替换为导入的任务
  - **合并导入**：保留现有任务，添加导入的任务

### 2. 修改 ManagementPanel 组件
- 文件：`src/components/ManagementPanel.tsx`
- 在侧边栏底部新增导入和导出按钮区域
- 新增 props：
  - `onExportTasks`: 导出任务的处理函数
  - `onOpenImportDialog`: 打开导入弹窗的处理函数
- UI 改进：
  - 在侧边栏底部添加固定区域显示导入导出按钮
  - 使用图标 + 文字的形式（小屏幕只显示图标）
  - 导出按钮：白色背景
  - 导入按钮：浅玫瑰色背景

### 3. 修改 App.tsx
- 添加导入弹窗状态管理：`isImportModalOpen`
- 添加 `handleOpenImportModal` 函数：打开导入模式选择弹窗
- 添加 `handleSelectImportMode` 函数：处理导入模式选择
- 将文件输入框移至 App 组件顶部，统一管理
- 传递导入导出相关函数到 ManagementPanel

### 4. 修改 AIConfigModal 组件
- 移除内部的文件输入框（已移至 App.tsx）
- 保留数据管理标签页中的导入导出按钮（用于配置页面访问）

## 用户体验改进

1. **更直观的访问方式**：
   - 在侧边栏底部常驻显示导入导出按钮
   - 用户无需进入配置页面即可快速导入导出任务

2. **清晰的导入流程**：
   - 点击导入按钮后弹出模式选择弹窗
   - 用户可以明确选择覆盖或合并模式
   - 每个选项都有详细说明，避免误操作

3. **一致的视觉风格**：
   - 使用项目统一的设计语言
   - 玫瑰色主题配色
   - 圆角卡片和按钮样式

## 使用方法

1. **导出任务**：
   - 打开管理面板（左侧边栏）
   - 点击底部的"导出"按钮
   - 任务数据将以 JSON 格式下载到本地

2. **导入任务**：
   - 打开管理面板（左侧边栏）
   - 点击底部的"导入"按钮
   - 在弹出的窗口中选择导入模式：
     - **覆盖导入**：清空现有任务，使用导入的数据
     - **合并导入**：保留现有任务，添加导入的任务
   - 选择 JSON 文件完成导入

## 技术细节

- 使用 Framer Motion 实现流畅的动画效果
- 使用 Lucide React 图标库
- 响应式设计，适配移动端和桌面端
- TypeScript 类型安全
- 与现有导入导出逻辑完全兼容
